# Travel-PA · Agent Intelligence

A calm, paper-white agent console for travel agents to search across thousands of bookings instantly.

## Features

- 🎯 Intelligent booking search
- 📊 Real-time agent status monitoring
- 🎨 Minimalist, accessible UI design
- 🚀 Fast, lightweight static site

## Hosting

This project is optimized for **Vercel** deployment.

### Quick Deploy to Vercel

1. Push your changes to GitHub
2. Visit [vercel.com](https://vercel.com)
3. Import this repository
4. Vercel will automatically detect and deploy your static site

### Local Development

```bash
# Using Python
python -m http.server 3000

# Or using Node.js http-server
npx http-server -p 3000
```

Then open `http://localhost:3000` in your browser.
